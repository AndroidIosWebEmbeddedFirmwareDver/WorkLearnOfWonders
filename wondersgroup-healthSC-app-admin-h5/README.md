### healthSC-app-admin-h5

---

### 产品简介

四川微健康后台管理H5

---

### 项目启动

a.下载依赖包
```
npm install
```

b.开发启动项目
```
npm start
```

c.访问地址
```
http://127.0.0.1:3004/healthSC-app-admin-h5/admin
name: admin
psd: 123456
```
