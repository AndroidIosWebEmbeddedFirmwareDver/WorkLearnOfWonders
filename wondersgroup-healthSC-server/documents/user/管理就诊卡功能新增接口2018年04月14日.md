##用户在线挂号就诊卡管理相关接口

###0.卡类型字典
~~~
{
    01//院内就诊卡
    02//居民健康卡
}
~~~

###1.新增卡
url
~~~
/api/user/online/medcial/cards/create
~~~
入参：
~~~
{
    "uid":"",//用户唯一编码ID
    "hospital_id":"",//医院ID；如果卡类型是就诊卡则不为空
    "hospital_code":"",//医院代码；如果卡类型是就诊卡则不为空
    "hospital_name":"",//医院名称；如果卡类型是就诊卡则不为空
    "mediacl_card_no":"",//在线挂号卡卡号
    "card_type_code":"",//在线挂号卡类型编码描述
    
}
~~~
出参：
~~~
{
    "code":"",//处理编码，1001-成功，1002-失败
    "msg":"",//处理消息
    "data":{
        "id":"",//在线挂号卡类型唯一ID
        "uid":"",//用户唯一编码ID
        "hospital_id":"",//医院ID；如果卡类型是就诊卡则不为空
        "hospital_code":"",//医院代码；如果卡类型是就诊卡则不为空
        "hospital_name":"",//医院名称；如果卡类型是就诊卡则不为空
        "mediacl_card_no":"",//在线挂号卡卡号
        "card_type_code":"",//在线挂号卡类型编码描述
    },//返回数据
}
~~~
